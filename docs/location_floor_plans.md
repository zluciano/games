## MINIMAP DISPLAY

The minimap shows a simplified version of current location:
- Green = walkable area
- Brown = paths/doors
- Blue = water
- Gray = buildings/walls
- Red dot = player position
- Yellow dot = exit points

```
+-------------+
| COURTYARD   |
|  +--+       |
|  |  |  *    |  <- * = player
|  +--+       |
|    |    o   |  <- o = exit
|    o        |
+-------------+
```
